# Taskmate

Taskmate is a task management web application built with Django. 
It allows users to register, log in, create, update, complete, and manage their daily tasks.

## Features
- User authentication (Register/Login/Logout)
- Create, edit, delete, and complete tasks
- Task status management
- Admin dashboard
- PostgreSQL database integration
- Responsive UI
- Deployed on Railway

## Tech Stack
- Python
- Django
- PostgreSQL
- HTML/CSS
- Bootstrap
- Gunicorn
- Railway
